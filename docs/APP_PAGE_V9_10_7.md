Criar a página oficial `/app/` para explicar e vender o app SimpleWay Academy sem cair em rota externa/404.

## Decisões
- CTAs “Acessar app” apontam para `/app/`.
- A página usa copy publicitária e didática, sem linguagem técnica pesada.
- Os “prints” são mockups visuais em CSS até existirem screenshots oficiais aprovados do app.
- A lista de testes envia mensagem para o WhatsApp `+55 19 99893-0846`.
