# QA Responsivo — SimpleWay Academy Site

## Mobile obrigatório

Testar principalmente:

- 360px — Android pequeno
- 390px — iPhone padrão
- 430px — iPhone Pro Max / Android grande

Checklist:

- Header não quebra.
- Menu mobile abre via botão.
- Botões ocupam 100% no mobile.
- Cards viram 1 coluna.
- Formulário vira 1 coluna.
- Hero não estoura horizontalmente.
- Títulos não cortam palavras importantes.
- Footer respeita safe-area.
- Não há scroll horizontal.

## Tablet

- 768px vertical
- 1024px horizontal

Checklist:

- Grids de cards com espaçamento confortável.
- CTA visível sem poluição.
- Formulário legível.

## Desktop

- 1366px
- 1440px+

Checklist:

- Hero premium.
- Cards alinhados.
- Copy escaneável.
- SEO pages navegáveis.
