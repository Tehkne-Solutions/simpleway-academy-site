# SimpleWay Academy — Site Institucional

Site institucional da SimpleWay Academy / SimpleWay English em Astro, preparado para Cloudflare Pages.

## Decisão de arquitetura

- Site institucional: Astro + Cloudflare Pages.
- Blog/SEO: dentro do mesmo site.
- Formulários: mensagem direta para WhatsApp.
- Plataforma oficial de aprendizagem: app SimpleWay Academy separado.
- Assinatura: Tehkné Solutions.

## Identidade

- Azul Elétrico: `#0057D8`
- Vermelho Vital: `#E63946`
- Dourado Divino: `#FFD166`
- Branco/Light premium como base visual
- Tipografia: Poppins + Inter
- Logo oficial em `public/brand/`

## Comandos

```bash
npm install
npm run build
npm run preview
```

## Cloudflare Pages

Configuração recomendada:

```txt
Framework preset: Astro
Build command: npm run build
Build output directory: dist
Deploy command: vazio
Node version: 20
```

No front comercial, usar:
