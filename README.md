# SimpleWay Academy — Site Institucional V9

Site institucional estático para Cloudflare Workers/Assets.

## V9

Front limpo, método visual rico, blog editorial sem metadados técnicos expostos e página de contato convertida em máquina de venda consultiva.

## Deploy Cloudflare

- Node version: 22
- Build command: `npm run build`
- Deploy command: `npm run deploy`
- Build output directory: `dist`
