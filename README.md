# SimpleWay Academy — Site Institucional V8

Site institucional premium, estático e preparado para Cloudflare Workers Assets.

## Marcador

`SWA_WORKER_DEPLOY_FIX_V8_CONTENT_SEO_PREMIUM`

## Entregas

- Header sticky com logomark oficial + texto lateral.
- Home light premium com animações, hover effects e imagens via Unsplash.
- 12 posts de blog com mínimo de 3.000 palavras cada.
- Single post colorida, educativa e com tabela de conteúdo sticky.
- Páginas institucionais expandidas.
- Painel editorial estático `/admin-conteudo/` para camada futura de curadoria.
- SEO técnico: canonical, Open Graph, Twitter Card, Schema.org, sitemap, robots, headers.
- Mobile-first e responsivo.

## Cloudflare

- Node version: 22
- Build command: `npm run build`
- Deploy command: `npm run deploy`
- Build output directory: `dist`
- Root directory: `/`
